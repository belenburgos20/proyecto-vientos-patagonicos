# Proyecto Vientos Patagónicos

## Descripción
Vientos Patagónicos es una empresa especializada en ofrecer servicios para zonas rurales, incluyendo la instalación de alambrados perimetrales y olímpicos, construcción de tanques australianos, movimiento de suelos, y suministro de materiales confiables. Nuestro objetivo es proporcionar soluciones efectivas adaptadas a las necesidades específicas de cada cliente y terreno.

## Características
- Compromiso con la calidad y la satisfacción del cliente.
- Uso de materiales de alta calidad.
- Equipo experimentado y profesional.
- Atención personalizada para cada proyecto.
- Experiencia en proyectos rurales y energías renovables (eólicos y solares).

## Servicios
- Instalación de alambrados de 5 hilos y olímpicos.
- Construcción de tanques australianos para almacenamiento de agua.
- Movimiento de suelos: nivelación, limpieza y preparación de terrenos.
- Suministro de materiales para obras rurales.
- Mantenimiento y reparación de instalaciones realizadas.

## Herramientas y Tecnologías
- **HTML5 y CSS3**: Para el diseño y estructura del sitio web.
- **Bootstrap 5**: Framework utilizado para el diseño responsivo y componentes.
- **JavaScript**: Para interactividad y funcionalidades dinámicas.
- **Google Maps API**: Integración de mapas interactivos.
